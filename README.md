# AgroApp 🌱

Aplicación móvil para el control de producción agropecuaria, desarrollada
en Flutter.

## Descripción

AgroApp permite a productores agropecuarios registrar y dar seguimiento
a sus parcelas, cultivos, riegos y fertilizaciones desde su dispositivo
móvil, incluso sin conexión a internet.

## Funcionalidades

- 🗺️ Registro de parcelas con fotografía y ubicación GPS
- 🌱 Registro de cultivos con seguimiento visual de etapa de desarrollo
- 💧 Control de riego con recordatorios programados
- 🧪 Control de fertilizantes
- ☁️ Alertas climáticas (heladas, lluvia fuerte, sequía)
- 📊 Gráfico de consumo de agua por mes

## Tecnologías utilizadas

- Flutter / Dart
- SQLite (sqflite) — base de datos local
- Geolocator — geolocalización GPS
- Image Picker — captura de fotografías
- flutter_local_notifications — notificaciones locales
- Open-Meteo API — pronóstico climático
- fl_chart — gráficos

## Instalación (APK)

1. Descargar el archivo APK desde la sección [Releases](https://github.com/cancapaccuchamaalexander-ui/AgroApp/releases)
2. Transferirlo al dispositivo Android
3. Permitir instalación desde "orígenes desconocidos"
4. Instalar y conceder los permisos solicitados
